
## Abstrakt {.unlisted .unnumbered}

Cílem diplomové práce je navrhnout a implementovat asynchronní knihovnu
pro vytváření datových toků v programovacím jazyce Rust.
Práce se zaměřuje na vytvoření knihovny,
která umožňuje typově bezpečnou tvorbu a kompozici toků.
Na základě analýzy principů programování datových toků,
asynchronního modelu jazyka Rust a souvisejících technik,
jako je polymorfismus, monomorfismus a práce s generickými typy,
je navržená architektura knihovny,
včetně kontextového systému pro sdílení dat mezi uzly.
Následně je implementována a otestována.
Součástí práce je testování funkčnosti knihovny a experimenty
pro vyhodnocení ovlivnění výkonu a škálovatelnosti toků.

**Klíčová slova:** Rust, toky, kompozice, vizualizace, asynchronní programovaní,
uzel, pipeline, workflow, runtime, framework, knihovna, crate, typová bezpečnost


## Abstract {.unlisted .unnumbered}

**Title:** Design and implementation of an asynchronous library for creating data flows

The goal of this diploma thesis is to design
and implement an asynchronous library for creating data flows
in the Rust programming language.
The thesis focuses on creating a library that enables type-safe creation and composition of data flows.
It is based on an analysis of data flow programming principles,
the asynchronous model of the Rust language,
and related techniques such as polymorphism, monomorphism,
and working with generic types.
Based on the results of the analysis, a library architecture is proposed,
including a context system for sharing data between nodes,
and subsequently implemented and tested.
As a part of the thesis, the functionality of the library is tested
and experiments are conducted to evaluate the impact on the performance
and scalability of data flows.

**Keywords:** Rust, flow, data flow, composition, visualization, asynchronous, node, pipeline,
workflow, runtime, framework, crate, library, type safety
